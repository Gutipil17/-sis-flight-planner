# GutiPilot Flight Planner 4.0 — Copiloto OACI

## Dictado

El micrófono del teclado del iPhone realiza la transcripción. GutiPilot analiza
localmente el texto coloquial, lo convierte en campos técnicos y muestra una
vista previa. Esta interpretación no requiere internet.

La validación en línea de AIP, NOTAM o datos operacionales no está incluida
todavía porque requiere fuentes y servicios externos confiables.

## Constructor de casilla 10

Incluye selección guiada de:

- COM/NAV.
- Equipos avanzados y enlace de datos.
- Vigilancia.
- Capacidades PBN aprobadas.

Genera automáticamente:

- Casilla 10 antes del slash.
- Casilla 10 después del slash.
- PBN/ para la casilla 18.

Configuración inicial del helicóptero actual:

S/CB1
