# GutiPilot Flight Planner 3.5 — Dictado con teclado del iPhone

## Cambios

- Se eliminó la dependencia del reconocimiento de voz de Safari.
- El piloto toca el cuadro de texto y usa el micrófono del teclado del iPhone.
- Se añadió una guía visual que muestra dónde está el ícono del micrófono.
- El texto dictado se analiza y muestra en vista previa antes de aplicarse.
- Se corrigió la aeronave activa para que muestre la última matrícula o HK3911.
- “Crear manualmente” ahora desplaza directamente al formulario.
- En Perfil se agregó el equipo de vigilancia de la casilla 10:
  - CB1 predeterminado para el helicóptero actual.
  - C, S, A, N.
  - Código personalizado.
- La cabecera muestra versión 3.5.
