# GutiPilot Flight Planner 3.2 — Crear plan por voz

## Novedades

- Botón “Crear plan por voz”.
- Leyenda didáctica con los datos recomendados para un plan completo.
- Reconocimiento de matrícula, salida, destino, DEP/, DEST/, fecha, hora, ruta,
  altitud, velocidad, EET, autonomía, POB, alterno, piloto, licencia y observaciones.
- Vista previa antes de aplicar.
- Detección de información faltante.
- Advertencias para datos críticos.
- Botón “Corregir por voz”.
- Transcripción editable manualmente.
- La aplicación nunca aplica el dictado sin confirmación del piloto.

## Compatibilidad

El reconocimiento automático depende del navegador. Cuando no esté disponible,
el usuario puede escribir el dictado en el mismo cuadro y usar “Analizar dictado”.
El resto de la aplicación continúa funcionando sin conexión después de cargarse.
