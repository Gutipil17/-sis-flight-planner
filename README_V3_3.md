# GutiPilot Flight Planner 3.3

## Voz con contexto activo

- Ya no exige dictar matrícula, piloto, operador, equipo ni datos operacionales fijos.
- Muestra antes del dictado los datos que ya están cargados.
- Solo solicita los datos variables que falten.
- Permite cambiar opcionalmente aeronave, VFR/IFR, tipo de vuelo o turbulencia mediante voz.
- Conserva la última aeronave seleccionada como aeronave activa.
- Mantiene vista previa y corrección por voz antes de aplicar.

## Perfil operacional editable

En Perfil pueden definirse como predeterminados:

- Reglas de vuelo: V, I, Y o Z.
- Tipo de vuelo: N, S, G, M o X.
- Categoría de turbulencia: L, M, H o J.

Estos valores se cargan en cada plan nuevo, pero pueden cambiarse temporalmente en el formulario.
