# PUBLICACIÓN RÁPIDA

Esta aplicación es completamente estática.

## Opción Netlify Drop
- Entre a Netlify Drop desde un computador.
- Arrastre la carpeta completa `SIS_Flight_Planner_Mobile`.
- Netlify entrega una dirección HTTPS.
- Abra esa dirección en Safari del iPhone y añádala a la pantalla de inicio.

## Opción GitHub Pages
- Cree un repositorio.
- Suba todos los archivos de la carpeta.
- Active Pages desde la rama principal.
- Abra la dirección HTTPS resultante en Safari.

No publique la carpeta dentro de otra carpeta adicional si desea que la aplicación abra desde la raíz.
