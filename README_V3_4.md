# GutiPilot Flight Planner 3.4 — Copiloto de voz

## Cambios principales

- El acceso por voz es ahora la primera opción al crear un plan.
- Pantalla inicial con:
  - Dictar el plan.
  - Crear manualmente.
  - Duplicar el último vuelo.
- El modal de voz se corrigió para abrir siempre.
- Los scripts se cargan después de toda la interfaz para evitar fallos de inicialización.
- Reconocimiento Web Speech cuando está disponible.
- Respaldo para iPhone: usar el micrófono del teclado dentro de la transcripción.
- Contexto activo con aeronave, piloto, operador y configuración.
- Vista previa obligatoria.
- Corrección por voz o edición manual.
- Nunca se aplican datos sin confirmación.
