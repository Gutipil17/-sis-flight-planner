# GutiPilot Flight Planner 3.0.5 — versión de pruebas

Correcciones incluidas:

- Historial protegido contra datos dañados y carga segura de planes anteriores.
- “Duplicar datos” restaura el plan y actualiza la fecha al día actual.
- Compatibilidad reforzada para evitar “File Error” al generar, compartir o guardar PDF en Safari/iPhone.
- Separación de datos: piloto al mando, licencia del piloto al mando y copiloto / “Presentado por”.
- Carepa (SKLC): oficina receptora Rionegro (SKRG).
- Pasto (SKPS): oficina receptora Cali (SKCL).
- Cali: 602-418-5080 para planes con salida desde el aeropuerto de Cali; 602-418-5124 para gestión de Regional Cali cuando la salida no es desde Cali.
- Directorio corregido para buscar por código, ciudad, aeropuerto, departamento y regional.

Esta compilación se identifica como 3.0.5 y usa una nueva caché del service worker.
